Android

When setting up an Android device with an emnify SIM, you need to follow these 4 steps:

  1. Go to Settings → Mobile network
  2. Go to Mobile data
  3. Enable Data roaming and go to Access Point Names (APNs)
  4. Create a New APN with any name and configure the APN with em
